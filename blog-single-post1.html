Para usurários de IOS e Android

Para a melhor experiência durante as visitas no site Cefhub, recomendamos o uso de um computador Mac ou Windows. Caso não tenha acesso aos mesmos, os dispositivos IOS serão eficazes mas os dispositivos android serão limitados por não terem acesso directo aos conteúdos pdfs encontrados no site. Mas se o site for aberto pela chrome (Google chrome), os usuários dos dispositivos android terão acesso aos documentos PDF encontrados no site.
